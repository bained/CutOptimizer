# QA Checklist for CutOptimizer

## Цветова палитра (Industrial Theme)
- [ ] Основен фон: `#F4F4F4` (Light Gray) / `#2C3E50` (Dark Navy - за заглавни части)
- [ ] Контраст: Използвай WCAG 2.1 стандарт (минимум 4.5:1 за текст).
- [ ] Акценти: `#3498DB` (Industrial Blue) - само за активни елементи/бутони.
- [ ] Валидация: Бяло за текстови полета, сиво за рамки.

## UX/UI Контролен списък
- [ ] **Usability:** Логичен ли е потокът от данни?
- [ ] **Layout:** Подредено ли е всичко по логическа йерархия?
- [ ] **Accessibility:** Ярки ли са шрифтовете? Четеми ли са при ниска светлина?
- [ ] **Feedback:** Дават ли диалоговите прозорци ясна информация при грешка?
- [ ] **Consistency:** Имат ли всички бутони еднаква форма/цвят?

## Процедура за ревю
1. Cline преглежда UI спрямо този файл.
2. Cline генерира списък с подобрения.
3. Потребителят одобрява промените.
