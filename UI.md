# Потребителски интерфейс

## Технологии

- **HTML5** — структура
- **CSS3** — стилизация (без frameworks)
- **Vanilla JS (ES6 modules)** — логика на UI
- **Neutralino.js** — OS интеграция (файлов диалог, прозорци)
- **SVG (inline)** — визуализация на резултатите

## Структура на UI

### 2. Панели

#### Input Panel (ляв панел)
- **Sheet Settings**: W, H, Kerf (числови полета)
- **Parts Table**: таблица с колони Име, W, H, Can Rotate (checkbox)
  - Бутон "Add" — добавя нов ред
  - Бутон "Delete" — премахва избран ред
  - Редакция на клетки (inline edit)
- **Algorithm Parameters**:
  - Итерации (числово поле, default: 1000)
  - Beam Width (числово поле, default: 12)

#### Visualization Panel (десен панел)
- **SVG Viewer**: inline SVG с всички листове
  - Всеки sheet е отделен правоъгълник с бял фон и червена рамка
  - Частите са цветни правоъгълници (синьо = нормални, зелено = ротирани)
  - Надписи: име на част + "[R]" ако е ротирана
  - Надписи за sheet: "Sheet N — Efficiency: X.X%"
- **Резултати**: брой листове, средна ефективност, време за изпълнение

### 3. Диалогови прозорци (Neutralino)

- **Open Project**: файлов диалог за .json
- **Save Project**: записване на текущия проект
- **Export SVG**: експорт на визуализацията
- **Settings**: конфигурация (config.ini)
  - iterations
  - current_json (последен отворен проект)

## Потребителски сценарии

1. **Нов проект**:
   - Въвеждане на W/H/Кerf
   - Добавяне на детайли (име, размери, ротация)
   - Run → Visualize

2. **Отваряне на проект**:
   - File → Open → избор на .json
   - Попълване на полетата от JSON
   - Run → Visualize

3. **Редакция**:
   - Добавяне/премахване на детайли
   - Промяна на размери
   - Run Again → актуализиране на визуализацията

## CSS Дизайн

- **Тъмна тема** (dark mode) по подразбиране
- Flexbox/Layout: `display: flex` за двата панела
- Минималистичен, индустриален стил
- SVG визуализация: responsive `viewBox`
- Скролване при много листове (overflow-y: auto)
